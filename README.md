### 게시물 작성

(작성자 기능)

- 프로젝트 제목

- 프로젝트 설명

- 모집인원  -> ★모집인원 : 2명

- 모집분야

  - 분야별 -> 개발자분야, 주요업무, 자격요건, 우대사항

  - 추가버튼을 눌러서 분야 추가

  - 자유롭게 쓸수  있는 공간 -> 아무내용이나

  

- 팀장 연락처(전화번호, 오픈채팅, 메일)

- 마감일(년/월/일/시/분)

- 작성자 정보 -> 회원가입시 작성한 내용 뿌려주기

- 댓글 기능 -> 작성, 삭제, 수정

- 마감버튼 -> 버튼을 누르면 "진행중"에서 "마감"으로 바뀌도록

(사용자 기능)

- 댓글 기능 
- 즐겨찾기 기능
